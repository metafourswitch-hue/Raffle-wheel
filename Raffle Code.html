Paste names, start the raffle, then hide this control panel for projector mode.

Blank lines are ignored. Duplicate names are kept unless you click “clean names,” which removes exact duplicates.
The app picks music automatically by elimination percentage. Level 1 plays early; Level 6 plays when only the final few remain. If a level is empty, it uses the closest earlier loaded level.
If no winner fanfare is selected, the elimination fanfare is used for the final winner.
Shortcuts: H = hide/show controls, F = fullscreen, Esc = exit fullscreen. The raffle runs automatically until one winner remains.
